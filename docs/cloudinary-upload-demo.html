📋 Payload tham khảo

Sau khi có imageUrl và imagePublicId từ Cloudinary, gửi chúng cùng các field khác:

Tạo topic mới — POST /topic 
{
  "name": "JavaScript Basics",
  "slug": "javascript-basics",
  "courseId": "<courseId>",
  "imageUrl": "https://res.cloudinary.com/...",
  "imagePublicId": "topic-images/javascript-basics"
}
Cập nhật ảnh — PUT /topic/:id 
{
  "imageUrl": "https://res.cloudinary.com/...",
  "imagePublicId": "topic-images/javascript-basics-v2"
}

// Ảnh cũ sẽ tự động bị xóa
// trên Cloudinary nếu publicId khác.

⚠️ imageUrl và imagePublicId phải được gửi cùng nhau hoặc bỏ cả hai. Gửi một trong hai sẽ bị báo lỗi 400 Bad Request.

🖼️ Topic Image — Signed Upload Flow

Step 1 — Lấy upload signature

MethodPOST
URL/topic/upload/signature
AuthBearer token — admin
Body (JSON){ "publicId"?: "string" } — tuỳ chọn, dùng để đặt tên file cố định
Response data: 
{
  "signature":    "abc123...",     // Chữ ký để xác thực upload
  "timestamp":    1718000000,      // Unix timestamp
  "folder":       "topic-images",  // Folder trên Cloudinary (từ env CLOUDINARY_TOPIC_IMAGE_FOLDER)
  "apiKey":       "your_api_key",
  "cloudName":    "your_cloud",
  "resourceType": "auto",
  "uploadUrl":    "https://api.cloudinary.com/v1_1/your_cloud/auto/upload"
}

Step 2 — Upload ảnh lên Cloudinary (client-side)

Gửi multipart/form-data trực tiếp tới Cloudinary — không qua server.

URLuploadUrl nhận từ Step 1
MethodPOST — multipart/form-data
fileFile ảnh cần upload
api_keyLấy từ signature response
timestampLấy từ signature response
signatureLấy từ signature response
folderLấy từ signature response
public_idpublicId bạn muốn đặt (phải khớp với lúc ký)
Lấy từ Cloudinary response: 
{
  "secure_url": "https://res.cloudinary.com/...",  // → imageUrl
  "public_id":  "topic-images/javascript-basics"   // → imagePublicId
}

Step 3a — Tạo topic mới với ảnh

MethodPOST
URL/topic
AuthBearer token — admin
Body 
{
  "name": "JavaScript Basics",
  "slug": "javascript-basics",
  "courseId": "<courseId>",
  "imageUrl": "https://res.cloudinary.com/...",
  "imagePublicId": "topic-images/javascript-basics"
}

Step 3b — Cập nhật ảnh cho topic có sẵn

MethodPUT
URL/topic/:id
AuthBearer token — admin
Body 
{
  "imageUrl": "https://res.cloudinary.com/...",
  "imagePublicId": "topic-images/javascript-basics-v2"
}
// ⚠️ Ảnh cũ sẽ tự động bị xóa khỏi Cloudinary

Validation rules

RuleChi tiết
imageUrl & imagePublicIdPhải gửi cùng nhau hoặc bỏ cả hai
imageUrl formatPhải là HTTPS URL thuộc res.cloudinary.com và đúng cloud name
Auto cleanupUpdate ảnh mới → ảnh cũ tự động bị xóa trên Cloudinary
Delete topicXóa topic → ảnh trên Cloudinary cũng bị xóa theo

📦 Course Project — Signed Upload Flow

Step 1 — Lấy upload signature

MethodPOST
URL/course/:id/upload/signature
AuthBearer token (user đã đăng nhập)
Body{ "publicId"?: "string" }

Step 2 — Upload file lên Cloudinary

Tương tự Topic — gửi multipart/form-data thẳng tới Cloudinary.

Định dạng hỗ trợ.zip, .rar, .pdf, .docx
Giới hạnTối đa 5 file, mỗi file tối đa 20 MB

Step 3 — Submit metadata về backend

MethodPOST
URL/course/:id/project-submission
Body 
{
  "note": "Em nộp bài lần đầu",
  "files": [{
    "secureUrl":    "https://res.cloudinary.com/...",
    "publicId":     "project-submissions/<courseId>/<userId>/my_project",
    "originalName": "my_project.zip",
    "mimeType":     "application/zip",
    "fileSize":     204800
  }]
}
